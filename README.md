# ВолонтёрГород

Веб-приложение для информирования жителей о волонтёрских инициативах и социальных проектах города.

## Описание проекта

«ВолонтёрГород» — это интерактивный информационный портал, который позволяет пользователям:

- Просматривать главную страницу с актуальными акциями и ближайшими мероприятиями
- Смотреть анимированные счётчики общей статистики волонтёрского движения
- Работать с каталогом проектов: фильтрация, поиск, сортировка, карточки статусов
- Открывать подробности проекта в модальном окне и записываться на участие
- Использовать интерактивную SVG-карту с точками мероприятий
- Просматривать таймлайн событий с фильтрами и диаграммой (Chart.js)
- Сохранять избранные проекты и заявки в localStorage
- Читать истории участников и смотреть фотоотчёты
- Изучать образовательный раздел о волонтёрстве

## Функциональность

### Каталог проектов

- Фильтрация по направлению: Экология, Дети, Пожилые, Животные, Городская среда
- Фильтрация по формату: Разовое, Регулярное, Онлайн
- Поиск по названию, описанию и ключевым словам
- Сортировка по дате, популярности, свободным местам
- Статусы проектов: «Набор открыт», «Мест мало», «Завершён»

### Карта мероприятий

- Собственная интерактивная SVG-карта районов
- Маркеры мероприятий с цветовой индикацией направлений
- Подсказки по наведению, модальные окна по клику
- Фильтры по направлению, формату и району
- Масштабирование карты

### Таймлайн

- Режим «Ближайшие» (список)
- Режим «По месяцам» (диаграмма Chart.js)
- Фильтрация по направлению и формату

### Моё волонтёрство

- Форма регистрации волонтёра с клиентской валидацией
- Список избранных проектов
- Список поданных заявок
- Удаление заявок и избранных
- Сохранение данных в localStorage

## Технологии

- HTML5 с семантической разметкой
- CSS3 (CSS Grid, Flexbox, CSS-переменные)
- JavaScript (ES6+) без фреймворков
- localStorage для пользовательских данных
- Chart.js для визуализации статистики и таймлайна
- Schema.org разметка

## Запуск проекта

1. Клонируйте репозиторий:

```bash
git clone git@github.com:Kagutsuchi-106/catalog.git
cd catalog_art
```

2. Откройте `index.html` в браузере.

## Данные

Проект использует JSON-данные в `scripts/data.js`:

- 22 волонтёрских проекта/мероприятия
- 8 историй участников
- 9 площадок/локаций
- 5 направлений помощи
